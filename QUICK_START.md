# FXBoard - 사용 가이드 (완성!)

## ✅ 이제 제대로 작동합니다!

### 🎹 키보드로 소리 재생하기

기본적으로 다음 키들이 활성화되어 있습니다:

```
활성화된 키 (파란색으로 표시):
┌───┬───┬───┬───┐
│ A │ S │ D │ F │  ← 이 키들을 눌러보세요!
└───┴───┴───┴───┘
 ↓   ↓   ↓   ↓
kick snare hihat clap
```

### 🎵 테스트 방법

1. **프로그램 실행**
   ```bash
   ./build/FXBoard_artefacts/FXBoard
   ```

2. **Keyboard 탭 선택** (기본값)
   - 파란색 버튼들이 활성화된 키입니다
   - A, S, D, F 키가 파란색으로 표시됩니다

3. **키보드로 연주**
   - **A 키**: Kick 드럼 🥁
   - **S 키**: Snare 드럼 🎵
   - **D 키**: Hi-Hat 🎶
   - **F 키**: Clap 👏

4. **시각적 피드백**
   - 키를 누르면 버튼이 **보라색**으로 빛납니다
   - 소리가 재생됩니다

### 📊 상태 패널 보는 법

창 상단의 상태 패널에서 실시간 정보 확인:

```
┌──────────┬──────────┬──────────┐
│ Latency  │ CPU Load │ Samples  │
│  2.7 ms  │  15.3%   │    12    │
│ (초록색)  │ (초록색)  │ (파란색) │
└──────────┴──────────┴──────────┘
```

- **Latency < 10ms**: 초록색 = 매우 좋음 ✅
- **CPU < 50%**: 초록색 = 안정적 ✅
- **Samples**: 로드된 샘플 개수

### 🎨 색상 의미

#### 키 버튼
- **회색**: 비활성 (샘플 할당 안됨)
- **파란색**: 활성 (샘플 할당됨) - 사용 가능!
- **보라색 빛**: 키를 누르는 중

#### 상태 표시
- **초록색**: 정상 (좋음)
- **노란색**: 주의 (보통)
- **빨간색**: 경고 (나쁨)

### 🔧 설정하기

오른쪽 상단 **Settings** 버튼 클릭:

1. **Key Mapping 탭**
   - 새로운 키 추가
   - 샘플 변경
   - 매핑 제거

2. **Samples 탭**
   - 로드된 샘플 확인
   - 새 샘플 추가
   - Refresh로 목록 갱신

3. **Audio 탭**
   - 오디오 디바이스 선택
   - 버퍼 크기 조정 (레이턴시 조절)

### 🎵 샘플 목록

현재 로드된 샘플들:
- ✅ kick (킥 드럼)
- ✅ snare (스네어 드럼)
- ✅ hihat (하이햇)
- ✅ clap (박수)
- ✅ crash (크래시 심벌)
- ✅ tom (탐)
- ✅ click (클릭)
- ✅ rim (림샷)
- ✅ beep1, beep2, beep3, beep4 (비프음)

### 🚀 고급 기능

#### 백그라운드 모드
창을 닫아도 프로그램이 계속 실행됩니다:
- 키 입력 계속 감지
- 메뉴에서 다시 열 수 있음

#### 백그라운드로 시작
```bash
./build/FXBoard_artefacts/FXBoard --hidden
```

### 💡 팁

1. **빠른 반응**: Latency가 낮을수록 좋습니다
2. **안정성**: CPU 사용률을 50% 이하로 유지하세요
3. **커스터마이징**: Settings에서 원하는 키에 샘플 할당
4. **여러 샘플**: 최대 26개 키에 샘플 할당 가능
