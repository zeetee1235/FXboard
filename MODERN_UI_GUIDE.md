# FXBoard - 모던 UI 가이드

## 🎨 새로운 디자인 특징

### 다크 테마
- 세련된 다크 모드 인터페이스
- 눈의 피로를 줄이는 색상 구성
- 고대비 색상으로 가독성 향상

### 컴포넌트
1. **헤더 섹션**
   - 앱 이름 표시
   - Settings 버튼 (오른쪽 상단)

2. **상태 패널**
   - **Latency**: 오디오 지연시간 (ms)
     - 초록색: < 10ms (매우 좋음)
     - 노란색: ≥ 10ms (보통)
   - **CPU Load**: CPU 사용률
     - 초록색: < 50% (안정적)
     - 노란색: 50-80% (주의)
     - 빨간색: > 80% (과부하)
   - **Samples**: 로드된 샘플 수

3. **탭 인터페이스**
   
   #### Keyboard 탭
   - 3줄의 키보드 레이아웃 표시
   - 각 키 버튼 특징:
     - 키 이름 (Q, W, E, A, S, D 등)
     - 할당된 샘플 이름 (하단에 표시)
     - 활성화된 키는 파란색으로 표시
     - 키를 누르면 보라색으로 하이라이트
   
   #### Samples 탭
   - 로드된 모든 샘플 리스트
   - Add Sample 버튼으로 새 샘플 추가
   - Refresh 버튼으로 목록 갱신

## 🎹 사용 방법

### 키보드 연주
1. **Keyboard** 탭을 선택
2. 파란색으로 표시된 키들이 활성화된 키입니다
3. 키를 누르면:
   - 버튼이 보라색으로 빛납니다
   - 할당된 사운드가 재생됩니다

### 기본 키 매핑
```
Q W E R T Y U I O P  (16-25)
 A S D F G H J K L   (30-38)
  Z X C V B N M      (44-50)
```

현재 활성화된 키:
- **A (30)**: kick
- **S (31)**: snare  
- **D (32)**: hihat
- **F (33)**: clap

### 샘플 관리
1. **Samples** 탭으로 이동
2. 로드된 모든 샘플 확인
3. **Add Sample**: 새 샘플 추가 (추후 구현)
4. **Refresh**: 샘플 목록 새로고침

### 설정
1. 오른쪽 상단의 **Settings** 버튼 클릭
2. 키 매핑, 샘플, 오디오 설정 조정

## 🎨 색상 의미

### 버튼 상태
- **회색 (기본)**: 비활성 키
- **파란색**: 활성화된 키 (샘플 할당됨)
- **보라색 하이라이트**: 키를 누르는 중

### 상태 표시
- **초록색**: 정상/좋음
- **노란색**: 주의/보통
- **빨간색**: 경고/나쁨
- **하늘색**: 활성/선택

## ⌨️ 키보드 단축키

- **키보드 입력**: 직접 키를 눌러 사운드 재생
- **Tab/Shift+Tab**: 탭 간 이동
- **창 닫기**: 백그라운드 모드로 전환 (프로그램 계속 실행)

## 📊 성능 모니터링

상태 패널에서 실시간으로 확인:
- **Latency**: 낮을수록 반응이 빠름 (목표: <10ms)
- **CPU**: 낮을수록 시스템 부담이 적음
- **Samples**: 많을수록 다양한 소리 사용 가능

## 🎵 팁

1. **낮은 레이턴시**: 오디오 설정에서 버퍼 크기를 줄이세요
2. **안정적인 재생**: CPU 사용률이 80% 미만으로 유지하세요
3. **키 할당**: Settings에서 원하는 키에 샘플을 할당하세요
4. **백그라운드 실행**: 창을 닫아도 키 입력은 계속 감지됩니다
