# samples 디렉토리

이 디렉토리에 WAV 형식의 오디오 샘플을 배치하세요.

## 샘플 준비 방법

1. WAV 파일을 이 디렉토리에 복사
2. `config.json`에서 샘플 매핑 설정
3. 파일 이름은 영문과 숫자만 사용 권장

## 권장 샘플 설정

- **샘플레이트**: 48kHz (또는 44.1kHz)
- **비트 뎁스**: 16-bit 또는 24-bit
- **채널**: 모노 또는 스테레오
- **길이**: 짧을수록 메모리 효율적 (0.5초 ~ 5초 권장)

## 예제 샘플

다음과 같은 샘플을 준비하면 좋습니다:

- `kick.wav` - 킥 드럼
- `snare.wav` - 스네어 드럼
- `hihat.wav` - 하이햇
- `clap.wav` - 클랩
- `crash.wav` - 크래시 심벌
- `tom.wav` - 탐
